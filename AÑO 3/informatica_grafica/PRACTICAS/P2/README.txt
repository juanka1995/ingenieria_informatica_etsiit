#############  INSTRUCCIONES DE EJECUCION ################

 Opciones:
    ./practica2 --> Ejecuta el programa normalmente
    ./practica2 -c [ruta_fichero_ply] --> Carga un objeto PLY
    ./practica2 -r [ruta_perfil_ply_revolucion] -n [numero_segmentos] --> Carga un perfil de un fichero PLY y lo revoluciona n veces
    ./practica2 -b [ruta_perfil_ply_barrido] -n [número_segmentos] -l [longitud_barrido] --> Carga un perfil de un fichero PLY y lo barre hasta longitud con n segmentos

    -c : cargar
    -r : revolucionar
    -b : barrer
    -n : numero de segmentos (int)
    -l : longitud del barrido (float)

#############  INSTRUCCIONES DE USO ################

Tecla 1 --> Activa tetraedro
Tecla 2 --> Activa cubo
Tecla 3 --> Activa rombo
Tecla 4 --> Activa piramide
Tecla 5 --> Activa objetos PLY (PLY, Revolución o Barrido)
Tecla 6 --> Activa perfil por revolución
Tecla 7 --> Activa perfil por barrido
Tecla 0 --> No muestra ninguna figura

Tecla p --> Dibuja la figura actual en modo puntos
Tecla l --> Dibuja la figura actual en modo lineas/aristas
Tecla c --> Dibuja la figura actual en modo ajedrez
Tecla f --> Dibuja la figura actual en modo solido
Tecla h/H --> Muestra las instrucciones de uso por pantalla
